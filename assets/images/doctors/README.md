# Doctor Photos

วางรูปถ่ายครึ่งตัวของแพทย์แต่ละท่านในโฟลเดอร์นี้

## ไฟล์ที่ต้องการ

| ไฟล์ | แพทย์ | ใช้ใน |
|------|-------|-------|
| `doctor-1.png` | นพ. สมชาย ใจดี | index.html, doctors.html |
| `doctor-2.png` | พญ. สุดา มั่นคง | index.html, doctors.html |
| `doctor-3.png` | นพ. วิชัย เจริญสุข | index.html, doctors.html |

| `doctor-5.png` | นพ. ธนา ประเสริฐ | doctors.html |
| `doctor-6.png` | พญ. อรุณี สดใส | doctors.html |

## ข้อกำหนดรูปภาพ

- **รูปแบบ:** PNG (พื้นหลังโปร่งใส) หรือ JPG/WebP
- **ขนาดแนะนำ:** 400x500px ขึ้นไป (อัตราส่วน 4:5)
- **สไตล์:** รูปถ่ายครึ่งตัว (half-body portrait)
- **พื้นหลัง:** โปร่งใส (PNG) จะสวยที่สุด เพราะจะซ้อนบน gradient สีทอง
